## Quick Start

- Open `index.html` in the folder to preview.
- Or run a tiny local server from the repository root:

```bash
cd renders/foundation-phase/01-personal-bio-card
python -m http.server 8000
# open http://localhost:8000
```

## What’s inside

- `index.html` — the demo and markup
- `css/style.css` — styles & responsive rules
- `js/` — optional small scripts (if present)
- `assets/` — images & icons (add `screenshot.png` here)

## How to update the screenshot

1. Open the page in a browser and take a screenshot of the card.
2. Save it as `screenshot.png` (or `screenshot.webp`) into the `assets/` folder.
3. Commit the new image — the README will show the preview automatically on GitHub.

## Notes & Tips

- Use optimized WebP for faster loads.
- Swap colors in `css/style.css` or add CSS variables for themes.
- Small changes: copy the `01-personal-bio-card` folder into other projects.
